AI-Powered Job Market Intelligence & Resume Analyzer

Overview

An AI-powered application that analyzes job market trends, matches resumes with jobs, evaluates resumes against job descriptions, identifies missing skills, and generates learning recommendations.

Built using Python, Streamlit, SQLite, Sentence Transformers, Scikit-Learn, and ReportLab.

Features

* Job Market Analytics Dashboard
* Top Skills Analysis
* Top Locations Analysis
* Resume PDF Upload
* Resume Parsing
* Career Recommendation
* Resume Score Calculation
* Resume Feedback
* Job Matching using NLP
* Resume vs Job Description Match %
* JD Missing Skills Detection
* AI Learning Roadmap
* PDF Report Generation

Tech Stack

* Python
* Streamlit
* SQLite
* Pandas
* Scikit-Learn
* Sentence Transformers
* ReportLab

Installation

git clone <repository-url>
cd JobMarketIntelligence
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
